# Ansible Playbooks

This repository contains a collection of Ansible playbooks I use to manage and automate system tasks across Linux and Windows environments.

## Playbooks Included

- **windows11_update.yml**  
  Installs the latest Windows 11 updates using Ansibleâ€™s Windows modules.

- **windows11_shutdown.yml**  
  Shuts down a Windows 11 VM remotely from the Ansible control node.
  
- **ubuntu_update_upgrade.yml**

  Updates all packages and removes packages that are not needed on Ubuntu host.

  
- **docker_health_check.yml**

  Check all running docker containers on host and output stats
